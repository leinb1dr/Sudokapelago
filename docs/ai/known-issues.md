# Known Issues

No recurring runtime or debugging issues are documented yet.

## Watch Areas

- Sudoku generation, validation, puzzle state, and configuration rules are planned but not implemented.
- Archipelago support currently stops at constructing an `archipelago.js` client; connection lifecycle, session recovery, item handling, and location checks are not implemented.
- The app is client-only and currently has no backend, database, persistence layer, or server-side API.
