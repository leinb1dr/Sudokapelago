# Domain Glossary

| Term | Meaning in this repository | Status |
|------|----------------------------|--------|
| Sudokapelago | The project name for a configurable Sudoku website with Archipelago multiworld integration. | Confirmed |
| Sudoku board | The future playable puzzle surface. No grid or cell model exists yet. | Planned |
| Puzzle configuration | Future options for generating or selecting Sudoku puzzles. Specific options are not defined yet. | Planned |
| Archipelago | The multiworld randomizer ecosystem that Sudokapelago is expected to connect to through `archipelago.js`. | Confirmed |
| Multiworld session | A future live Archipelago session connection. No connection flow exists yet. | Planned |
| Item | A future Archipelago item concept. No Sudokapelago item mapping exists yet. | Planned |
| Location | A future Archipelago check/location concept. No Sudokapelago location mapping exists yet. | Planned |
